# Kanban-Dashboard
This is the UI for Kanban Dashboard for our College's mini-Project "TaskChef".

Added a bit of functionalities like : 
1) Countdown Timer
2) Servings update feature
3) Different Media screen adaptions
4) Light and Dark Mode
